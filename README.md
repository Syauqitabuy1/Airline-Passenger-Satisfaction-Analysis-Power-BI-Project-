# Airline Cleanliness & Customer Satisfaction Dashboard

## 📌 Project Overview
This project analyzes airline passenger satisfaction data, focusing on the impact of cleanliness on overall customer experience.

## 🎯 Objectives
- Understand how cleanliness affects passenger satisfaction
- Identify differences across passenger segments
- Provide actionable insights for service improvement

## 🛠 Tools Used
- Power BI
- Excel / CSV Dataset

## 📊 Key Insights
- Passengers who rated cleanliness low tend to have significantly lower satisfaction levels
- Business class passengers generally rate cleanliness higher than economy class
- Cleanliness is one of the key drivers of overall customer satisfaction

## 📁 Dataset
Dataset sourced from Kaggle: Airline Passenger Satisfaction

## 🚀 How to Use
Download the `.pbix` file and open it using Power BI Desktop to explore the dashboard interactively.
